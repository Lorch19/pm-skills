# Create an Asset

Generate polished, branded sales assets tailored to your prospect.

## Format Options

1. **Interactive Landing Page** — Personalized web page for prospect
2. **Deck-Style** — Slide-format presentation
3. **One-Pager** — Single-page summary/comparison
4. **Workflow Demo** — Architecture or process visualization

## Process

1. **Context** — Seller info, prospect, audience, purpose, format
2. **Research** — Adaptive based on context richness
3. **Structure** — Format and audience appropriate
4. **Content** — Section templates per format
5. **Design** — CSS color system, typography, components
6. **Build** — Self-contained HTML artifact
7. **Iterate** — Refinement support

## What I Need

- **Who is this for?** — Company, role, industry
- **What's the goal?** — Educate, convince, compare
- **Format preference** — Landing page, deck, one-pager, demo
- **Key messages** — What should they walk away with?
