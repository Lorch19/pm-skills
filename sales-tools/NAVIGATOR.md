# Sales Tools — Navigator

## Quick Decision Tree

### Research & Prepare
- Researching a company or person → `account-research`
- Preparing for a sales call → `call-prep`
- Morning briefing on today's priorities → `daily-briefing`
- Competitive landscape analysis → `competitive-intelligence`

### Execute & Communicate
- Writing cold/warm outreach → `draft-outreach`
- Creating sales assets (decks, one-pagers) → `create-an-asset`
- Summarizing a call with action items → `call-summary`

### Manage & Forecast
- Weekly pipeline review → `pipeline-review`
- Quarterly forecast with scenarios → `forecast`

## All Skills

| Skill | Use When |
|-------|----------|
| `account-research` | Researching a company/person before outreach |
| `call-prep` | Getting ready for any sales call |
| `call-summary` | Processing notes/transcript into action items + follow-up |
| `competitive-intelligence` | Building interactive battlecards |
| `create-an-asset` | Landing pages, decks, one-pagers for prospects |
| `daily-briefing` | Morning sales briefing with priorities |
| `draft-outreach` | Personalized cold/warm outreach messages |
| `forecast` | Weighted forecast with best/likely/worst scenarios |
| `pipeline-review` | Pipeline health, deal prioritization, risk flags |
