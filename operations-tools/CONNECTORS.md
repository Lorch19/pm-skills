# Connectors

Plugin files use `~category` as a placeholder for whatever tool the user connects in that category.

| Category | Placeholder | Examples |
|----------|-------------|---------|
| Calendar | `~calendar` | Google Calendar, Microsoft 365 |
| Chat | `~chat` | Slack, Microsoft Teams |
| Email | `~email` | Gmail, Microsoft 365 |
| ITSM | `~ITSM` | ServiceNow, Zendesk, Jira Service Management |
| Knowledge base | `~knowledge base` | Notion, Confluence, Guru |
| Project tracker | `~project tracker` | Asana, Jira, Linear, ClickUp |
| Procurement | `~procurement` | Coupa, SAP Ariba, Zip |
