# Operations Tools — Navigator

## Quick Decision Tree

**What are you trying to do?**

### Document & Standardize
- Writing an SOP or process doc → `process-doc`
- Creating an operational runbook → `runbook`
- Optimizing an inefficient process → `process-optimization`

### Plan & Assess
- Planning team capacity for next quarter → `capacity-plan`
- Evaluating a vendor or contract → `vendor-review`
- Identifying project/operational risks → `risk-assessment`

### Track & Report
- Generating a status report for leadership → `status-report`
- Tracking compliance/audit readiness → `compliance-tracking`

### Change Management
- Proposing a system or process change → `change-request`

## All Skills

| Skill | Use When |
|-------|----------|
| `capacity-plan` | Quarterly planning, headcount decisions, workload analysis |
| `change-request` | System/process changes needing approval, CAB review |
| `compliance-tracking` | Audit prep, SOC 2, ISO 27001, GDPR tracking |
| `process-doc` | Formalizing tribal knowledge into SOPs, RACI matrices |
| `process-optimization` | Streamlining slow/wasteful workflows |
| `risk-assessment` | Evaluating risks for projects, vendors, decisions |
| `runbook` | Step-by-step operational procedures for ops/on-call |
| `status-report` | Weekly/monthly updates with KPIs and risks |
| `vendor-review` | Vendor evaluation, contract renewal, TCO analysis |
