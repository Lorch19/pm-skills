# Design Tools — Navigator

## Quick Decision Tree

### Review & Evaluate
- Get feedback on a design → `design-critique`
- Accessibility audit → `accessibility-review`
- Audit/document design system → `design-system`

### Create & Document
- Developer handoff specs → `design-handoff`
- Write UI copy, error messages, CTAs → `ux-copy`

### Research & Synthesize
- Synthesize interview/survey data → `research-synthesis`
- Plan a research study → `user-research`

## All Skills

| Skill | Use When |
|-------|----------|
| `accessibility-review` | WCAG 2.1 AA audit on design or page |
| `design-critique` | Structured feedback on usability, hierarchy, consistency |
| `design-handoff` | Developer specs: layout, tokens, states, responsive, edge cases |
| `design-system` | Audit, document, or extend design system components |
| `research-synthesis` | Distill interviews/surveys into themes and recommendations |
| `user-research` | Plan studies, write interview guides, design surveys |
| `ux-copy` | Microcopy, error messages, empty states, CTAs |
